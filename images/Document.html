Mastermind is a puzzle game, in which one player tries to guess the code their opponent comes up with

The code maker select a code - a combination of colors

This is the code that the code breaker will try to guess (max 8 guesses).

Next to each "guess row" is a small square with enough holes for four tiny pegs.

These pegs only come in two colors: white and black and give clues about how good the guess was:

  • Each white peg means that one of the guessed pegs is correct, but is in the wrong hole.
  • Each black peg means that one of the guessed pegs is correct, and is in the right hole.
  • The order of the white and black pegs does not matter.

Continue until the code is guessed or there are no more guesses left. The code breaker continues to make guesses, using information from all the previous hints he earned. If he manages to guess the complete code in exactly the right order, he wins the game. If he fails to guess and fills every row with pegs, the code maker wins instead.
